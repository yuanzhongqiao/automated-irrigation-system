<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这是一个自动给植物浇水的开源应用程序。到目前为止，几乎没有免费的专业软件和说明可用于构建可扩展、准确且最重要的是耐用的 DYI 灌溉。该应用程序不仅是为了美观和对数据的热爱。最重要的是，它是一种根据植物的具体需求定制传感器的工具。这是大多数直接测量土壤湿度的灌溉系统失败的地方，因为每种土壤和植物都不同，因此手动校准以及可能在一段时间后重新校准是必要的。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">该应用程序包含以下功能：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">监控并展示分钟、小时、日、周、月级别的时间序列数据</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">设置应触发自动浇水的水位。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">设置灌溉期间泵的工作时间</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">通过按钮手动启动灌溉</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在不同的传感器配置文件之间切换</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">在深色和浅色主题之间切换</font></font></li>
</ul>

<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">选择该架构是为了使泵逻辑和测量数据的记录是分开的。这使得可以使用 Raspberry Pi 控制多达 26 个泵（默认可用 GPIO 引脚的数量）。用Raspberry本身也无法读取电容式传感器的模拟信号，因为Raspberry只能处理数字信号。当然，可以使用 MCP3008 和串行接口读取传感器，但这需要更多引脚，并且设置不像以前那么干净。泵还单独连接到电源，其电路由继电器控制。因此也可以使用12V或更高电压的泵。</font></font></p>
